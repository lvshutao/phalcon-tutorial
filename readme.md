## 开始

不要安装 `phalcon/devtools` 因为有 bug

```
data/ 数据目录
    |-- mysql
    |-- pgsql
    |-- redis
log/ 日志目录
    |-- mysql
    |-- nginx
    |-- php
php/
    |-- extra.ini PHP 配置文件
src/ 项目目录
```

如果使用 phpStorm IDE，可以安装 `phalcon 5` 插件。
